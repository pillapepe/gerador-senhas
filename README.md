# Gerador de Senhas

Gerador feito em Python que cria senhas aleatórias com base nas preferências do usuário.

## Como usar

1. Execute o arquivo `gerador_senhas.py`;
2. Informe o tamanho desejado da senha (máximo 128 caracteres);
3. Escolha quais tipos de caracteres incluir: letras, números e/ou símbolos;
4. O programa exibe a senha gerada.
